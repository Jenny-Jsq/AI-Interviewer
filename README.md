# AI Mock Interview Agent (Business School)

An AI-powered mock interview tool designed for business school applicants.

## 🎯 What This Project Does

This tool simulates realistic admissions interviews based on:

- Target school & program
- Interviewer persona (adcom / alumni)
- User resume & background

Users can:
- Practice interview questions
- Receive follow-up questions
- Get structured feedback & takeaways
- Copy improvement suggestions

⚠️ This is a **single-session tool** (no login, no data storage).
